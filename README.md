# LesCouple

## 工程目录

```js
├── api // 接口目录

├── components // 公用组件目录

├── constants // 公用常量目录

├── pages // 主包页面目录

├── pagesM // 红娘地带相关页面目录

├── pagesU // 设置、个人信息、订单相关页面目录

├── static // 静态资源

├── uni_modules // 插件

├── utils // 公共方法

├── main.js // 项目入口文件

├── .gitignore // 不纳入 git 版本控制的 /文件夹?/ 列表

├── pages.json // 项目信息文件

└── README.md // 说明
```
